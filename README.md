# linux 스터디

## 진행방식

- 당일 진도범위, 진행자를 결정한다(로테이션).
- 모두 진도범위를 끝낼때까지 공부를 한다. (단, 상황을 보고 일찍 끝낼 수 있다.)
- 진행자는 공부가 끝나면 정리한 내용을 설명한다.
- 진행자는 정리한 내용을 `branch`로 올리고 `main`으로 PR을 올린다.
- 진행자가 아닌 나머지 인원은 다음시간 전까지 PR 내용을 리뷰한다.
- 진행자는 다음발표 전까지 PR을 merge한다.


## 진행 순서

```
이종혁 → 정아현 → 박진성 → 진형준(미정)
```
