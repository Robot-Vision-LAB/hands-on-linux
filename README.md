# hands-on-linux

## 진행방식

- 당일 챕터에서 공부할 분량을 인원에 맞게 할당한다.
- 각자 맡은 챕터를 정리하고 `branch`를 만든 후 `PR`을 생성한다.
  - 명명 규칙은 `chapter{챕터 번호}_{맡은 파트 번호}_{이름}`으로 한다.
- 다른 스터디원을 리뷰어로 추가한다.
- 리뷰가 끝나면 `main`으로 머지한다.
