# 셸 변수 표시하기
현재 유효한 셸 변수를 조사하려면 declare 명령어을 사용

## 사용 예
현재 유효한 모든 셸 변수를 표시
```
$ declare
```

## 프롬프트 변경법
\로 시작하는 문자열의 특수한 의미
| 기호 | 의미 |
|---|---|
| \d | 날짜 |
| \h | 호스트명 |
| \n | 줄 바꿈 |
| \s | 셸 이름(bash) |
| \t, \T, \@ | 시간(24시간, 12시간, am/pm 형식) |
| \\! | 명령어 이력 번호 |
| \u | 사용자명 |
| \w | 현재 디렉터리명 |
