# 텍스트 파일 길이 조사하기

wc: 텍스트 파일 길이(줄 수, 단어 수, 문자 수)를 조사

~~~ shell
$ wc states
50 60 472 states
~~~

- 한국어 같은 멀티바이트 문자를 포함하면 바이트 수로 표시
- 문자 수에는 줄 바꿈 문자 같은 제어 문자도 포함

|주요 옵션 ||
|----|---|
|-l|줄 수|
|-w|단어 수|
|-c|문자 수. 멀티바이트 문자를 포함하면 바이트 수|
|-m|문자 수. 멀티바이트 문자를 포함해도 문자 수|

~~~shell
# last 명령어 실행 결과 조사
$ last | wc
~~~
