# 파일 권한 변경하기

- `chmod`: 현재 파일 권한을 변경

    - chmod [arg1] [arg2] : arg2에 지정한 파일 권한을 arg1에 지정한 모드 설정으로 변경

### 모드 설정

- 모드 설정 : 대상 + 설정 방법 + 허가 조건

    - 대상
    
        - 소유자 + 소유자 소속 그룹 + 기타
        
        - 소유자 : u, 소속 그룹: g, 기타: o

        - 여러 대상을 동시에 지정하거나 전부를 대상으로 지정 가능: a

        - 아무것도 지정하지 않으면 전부를 지정한 것과 같음

    - 권한 설정 방법

        - 현재 권한에 추가 : +

        - 현재 권한에 삭제 : -

        - 지정할 조건으로만 설정 : =

    - 허가 조건

        - 읽기 : r

        - 쓰기 : w

        - 실행 : x

        - 허가가 없는 항목 : -


### 8진수를 사용한 권한 지정법

|8진수| rwx|
|---|---|
|0|---|
|1|--x|
|2|-w-|
|3|-wx|
|4|r--|
|5|r-x|
|6|rw-|
|7|rwx|

> chmod [옵션] 모드설정파일

|옵션| 설명|
|---|---|
|-R| 서브 디렉터리 이하도 한꺼번에 변경|
