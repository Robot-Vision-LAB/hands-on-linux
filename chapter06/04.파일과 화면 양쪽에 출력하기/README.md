# 파일과 화면 양쪽에 출력하기

- `tee`: 표준 입력으로 입력받은 내용을 화면 (표준 출력)에 출력하면서 인수로 지정한 파일에 저장(덮어쓰기)

> tee [옵션] [파일...]

|옵션|설명|
|---|---|
|-a| 입력을 파일 끝에 추가|

```bash
# ls 명령어 출력 결과를 표준 입력으로 받아 화면(표준 출력)에 표시하면서 file 파일에 저장
$ ls | tee file

# ls 명령어 출력 결과를 표준 입력으로 받아 화면 (표준 출력)에 표시하면서 file 파일 끝에 추가
$ ls | tee -a file
```
